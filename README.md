# Project Management 

A Spring Boot REST API for managing Users, Projects and tasks with JWT Authentication

# How to Run
Extract the ZIP file for the project. Extract it and update the configurations to Java 17 

# Create DataBase
Create database project_management

# db credentials
Update the username and password of db accordingly

# tables
Tables will be auto created when you run the spring application successfully 

# Run app

mvn clean install
mvn spring-boot:run

# Endpoints

Will provide the Postman collection for the REST APIs used

Note: Update the JWT token while testing the APIs. Get the token from Login API
